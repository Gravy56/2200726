Setup Required

Card Vault needs two Google Cloud APIs to work. Follow these steps to get your keys — it takes about 5 minutes and both APIs have generous free tiers.

1
Go to console.cloud.google.com and create a new project called card-vault
2
Enable the Cloud Vision API and Google Sheets API from the API Library
3
Go to Credentials → Create API Key. Copy it into the Vision API Key field in the sidebar
4
Create a new Google Sheet, copy the Sheet ID from the URL (the long string between /d/ and /edit), paste it in the sidebar
5
For Sheets write access, go to Credentials → OAuth 2.0 → Create Web Client ID, then click Authorize Sheets below
